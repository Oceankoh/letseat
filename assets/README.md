# Assets

## App Icon

- Source icon: [`app-icon.svg`](./app-icon.svg)
- PNG exports: [`app-icon-192.png`](./app-icon-192.png), [`app-icon-512.png`](./app-icon-512.png)
- Concept: friendly lion eating from a bowl.
- Intended use: PWA icon, favicon source, and app launcher icon once the application scaffold exists.

When the frontend is added, use the PNG exports in the web app manifest and regenerate them from the SVG if the source changes.
